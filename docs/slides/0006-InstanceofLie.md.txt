# Как работает instanceof?

<div className='codesmall'>

```javascript
function foo() { };

const bar = { a: 'a' };

Object.setPrototypeOf(
    foo.prototype,
    bar    // ← прототип функции указывает на объект!
);

const baz = Object.create(foo.prototype);
```

</div>

<Box sx={{marginTop: '5vh'}}>
<center>
<h3 style={{color: 'gray'}}>Мы сделали цепочку: baz → foo.prototype → bar</h3>
</center>
</Box>

-----

# Запустим и посмотрим

<div className='codesmall'>

```javascript
console.log(baz instanceof foo);  // true

// Но baz — это просто объект созданный от bar!
// Через Object.create(foo.prototype),
// где foo.prototype.__proto__ === bar
```

</div>
