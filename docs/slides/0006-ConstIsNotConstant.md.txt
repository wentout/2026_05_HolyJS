# const &mdash; это про привязку, не про значение

<div className='codesmall'>

```javascript
let toBeProp = 5;
const myObject = {};

Object.defineProperty(myObject, 'prop', {
    get () {
        return { [Symbol.toPrimitive] () {
            // ← returns CURRENT toBeProp
            return toBeProp; 
        }};
    }
});

const { prop } = myObject;  // prop — НЕ число!
```

</div>

# память указателя на const изменяема !

<div className='codesmall'>

```javascript
let toBeProp = 5;
const myObject = {};

Object.defineProperty(myObject, 'prop', {
    get () {
        return { [Symbol.toPrimitive] () {
            // ← returns CURRENT toBeProp
            return toBeProp; 
        }};
    }

const { prop } = myObject;

// Сюрприз Сюрприз : "const" переменная изменила своё значение !:
```
