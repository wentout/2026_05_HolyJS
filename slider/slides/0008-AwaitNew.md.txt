# await new — что это значит?

<div className='codesmall'>

```javascript
class MyPromise extends Promise {
    constructor(handler) {
        super(handler);
    }
}

const myItem = new MyPromise((resolve) => {
    resolve(123);
});

console.log(await myItem);  // 123
```

</div>

<Box sx={{marginTop: '5vh'}}>
<center>
<h3 style={{color: 'gray'}}>Мы можем await-ить результат new!</h3>
</center>
</Box>
