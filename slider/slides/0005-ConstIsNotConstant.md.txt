# const - это про привязку, не про значение

<div className='codesmall'>

```javascript
let toBeProp = 5;

const myObject = {};
Object.defineProperty(myObject, 'prop', {
    get () {
        return {
            [Symbol.toPrimitive] () {
                return toBeProp;
            }
        };
    }
});

const { prop: prop1 } = myObject;
```

</div>

<Box sx={{marginTop: '5vh'}}>
<center>
<h3 style={{color: 'orange'}}>Getter — это Interface. Он может измениться.</h3>
</center>
</Box>
