# Асинхронные геттеры и сеттеры

<div className='codesmall'>

```javascript
const myObj = {};
let field = 123;

Object.defineProperty(myObj, 'field', {
    async get () {
        return new Promise((resolve) => {
            setTimeout(() => resolve(field), 1000);
        });
    },
    async set (value) {
        // ... async operation
        field = value;
    }
});
```

</div>

<Box sx={{marginTop: '5vh'}}>
<center>
<h3 style={{color: 'orange'}}>get/set — это Interface. Они могут быть асинхронными!</h3>
</center>
</Box>
