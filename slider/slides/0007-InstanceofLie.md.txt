# Как работает instanceof?

<div className='codesmall'>

```javascript
function foo() { };
const bar = { a: 'a' };

// Создаём фейковую цепочку прототипов:
Object.setPrototypeOf(foo.prototype, bar);
const baz = Object.create(foo.prototype);

// Проверим:
console.log(baz instanceof foo);  // true
// Но baz НИКОГДА не создавался через new foo()!
```

</div>

<div className="slide-callout">
<h3>Цепочка: baz → foo.prototype → bar</h3>
</div>

-----

# Запустим и посмотрим

<div className='codesmall'>

```javascript
// Та же цепочка:
function foo() { };  const bar = { a: 'a' };
Object.setPrototypeOf(foo.prototype, bar);
const baz = Object.create(foo.prototype);

// Ложь:
console.log(baz instanceof foo);  // true
// baz создан от bar, где foo.prototype.__proto__ === bar
```

</div>

# instanceof не проверяет тип

<div className='codesmall'>

```javascript
// instanceof НЕ спрашивает: "ты экземпляр foo?"
// instanceof спрашивает: "есть ли в цепочке прототипов
//  прототип с конструктором foo?"

// Это traversal algorithm, не type check!
// baz → foo.prototype → bar → Object.prototype → null
//          ↑  has .constructor === foo
```

</div>
