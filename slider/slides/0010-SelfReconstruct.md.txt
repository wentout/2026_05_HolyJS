# Как выглядит паттерн Self Reconstructor в JS

<div className='codemedium'>

```javascript
const SelfReConstructor = function () {
    const main = this.constructor;
    const isItSelf = main === SelfReConstructor;
    const Self = function () {
        if (new.target) {
            return new (isItSelf ?
                SelfReConstructor : main);
        }
        return Self;
    };
    Object.setPrototypeOf(Self, this);
    Self.prototype.constructor = this;
    return Self;
};
```

</div>
