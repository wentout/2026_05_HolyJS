# await new — class + result

<div className='codemedium'>

```javascript
class MyPromise extends Promise {
    constructor(handler) { super(handler); }
}

const myItem = new MyPromise((resolve) => { resolve(123); });

// Мы можем await-ить результат new!
console.log(await myItem);  // 123

// И проверить наследование:
console.log(myItem instanceof Promise);  // true
```

</div>

<div className="slide-callout">

# можно и без наследования от Promise

<div className='codemedium'>

```javascript
class MyAsyncClass {
	field = undefined;
	constructor() {
		const self = this;
		return new Promise((resolve) => {
			setTimeout(() => {
				self.field = 123;
				resolve(self);
			}, 1000);
		});
	}
}
```

</div>

-----

# можно и без наследования от Promise

<div className='codemedium'>

```javascript
class MySubAsyncClass extends MyAsyncClass {
	constructor() {
                // да, super возвращает результат ! 
		const promise = super();
		return new Promise(async (resolve) => {
			// тут item содержит  this !!!
			const item = await promise;
			console.log(item.field); // 123
			resolve(item);
		});
	}
}
```

</div>
